To Run Cineverse Type in Cinverse Terminal:
1. composer update
composer install
2. npm install (harus punya node js dulu)
3. npm run dev
4. Open localhost/phpmyadmin make database admin
5. type in cineverse new terminal: php artisan migrate
6. new terminal trs type: php artisan serve
7. in chrome open: http://127.0.0.1:8000/
8. try : http://127.0.0.1:8000/admin-dashboard
9. kalau udh jalan type in new cineverse terminal: npm install datatables.net-bs4
10. LIat kalau tablenya yg di http://127.0.0.1:8000/admin-dashboard jadi bagus brti all good
